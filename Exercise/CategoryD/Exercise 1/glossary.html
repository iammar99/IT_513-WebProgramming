Introduction to HTML

Why Learn HTML?

HTML stands for HyperText Markup Language and it is the backbone of all websites. It helps developers define the structure and meaning of content on the web.

HTML is the standard markup language for creating Web pages. It describes the structure of a Web page, and it consists of a series of elements.

Code Example


    <!DOCTYPE html>
    <html lang="en">
    <head>
      <meta charset="UTF-8">
      <title>Hello World</title>
    </head>
    <body>
      <h1>Hello, World!</h1>
    </body>
    </html>

To refresh the page, press Ctrl + R.
